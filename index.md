## 📖 About this application

This application creates a safe environment to learn Linux. You practice command line tools on your Windows computer without needing a virtual machine. The software mimics the behavior of a standard Linux terminal. You learn how to manage files, set permissions, and write basic shell scripts. This tool helps students and beginners build confidence with system administration and cybersecurity concepts.

## 🧠 Common tasks explained

### Working with directories
Use `ls` to view the contents of your current folder. Use `cd` to move into a different folder. For example, typing `cd documents` moves you into that specific folder. Use `pwd` if you get lost, as this command displays your current location on the virtual system.

### Creating and editing files
Use `touch` followed by a filename to create a new, empty file. Use the built-in editor to add text. Once you save your file, use `cat` to see the contents. This confirms that your command worked as expected.

### Changing file rights
Linux uses a system of permissions. Use `chmod` to change who has the right to read, write, or run a file. The application includes a specific guide that walks you through these numerical values so you understand how to control access.

## 📝 Frequently asked questions

Do I need to be a programmer?
No. This tool is for beginners. You do not need to read code or understand how to build software.

Is this a real Linux system?
No. This is a simulation. The commands work just like they do on a real Linux computer, but your actual Windows files remain untouched and safe.

What happens if I enter a dangerous command?
Since this is a simulated environment, entering a command like `rm -rf /` will not delete your files. The application responds but nothing changes on your physical computer.

Can I save my work?
The application saves your progress automatically. You can close the program and return to your lessons whenever you have time to study.
